Atividade 9

Descrição da Atividade

Esta atividade consiste em criar uma página web que descreva a atividade 9, utilizando HTML e CSS para estilização. A página deve conter um título principal, subtítulos e parágrafos de texto.

Objetivos

  • Praticar a criação de páginas web com HTML.
  • Aplicar estilos CSS para melhorar a aparência da página.
  • Organizar o conteúdo de forma clara e legível.
